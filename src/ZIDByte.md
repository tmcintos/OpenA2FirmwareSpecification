### <a id="zidbyte-fbc0"></a>ZIDByte ($FBC0)

**Description:**

This is one of the Monitor ROM's main identification bytes, a fixed hexadecimal value (`$E0`) located in the ROM. It serves to indicate that the system is an enhanced Apple //e or a later model. This is not a callable routine.

**Input:** None.

**Output:** None.

**Side Effects:** None.
